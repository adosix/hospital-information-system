Instalace

Stručně popište:
  • postup instalace na server,

    • Vytvoríme virtuálne prostredie python3. Aktivujeme virtuálne prostredie. Nainštalujeme mysqlclienta a prihlásime sa do mysql ako root. Vytvoríme databázu "hospital_data". Vytvoríme používateľa "djangouser" s heslom "password". Dáme mu všetky práva na databázu "hospital_data". Nainštalujeme django a do neho inštalujeme django-multiforloop a django-multiforloop. presunieme sa do priecinku kde sa nachádza skript start.sh a spustíme ho, týmo sa vykonajú migrácie a spustí sa server na porte 8080

  • softwarové požadavky (verze PHP apod.),

    • php 7.4.3 python 3.8.5 pip3 django mysqlclient

  • jak rozbalit, konfigurovat, inicializovat databázi, ...

    • Inicializujeme databázu tak, že z priečinku db zavoláme skript db_setup.sh.
